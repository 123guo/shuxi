# 数栖平台

​		数栖平台提供从数据同步、数据开发到数据管理的一站式大数据全链路解决方案，支持Hadoop、Spark、Flink、Presto等业界流行的大数据计算引擎，并支持快速扩展。通过此平台可构建PB级别的数据仓库，来实现超大规模数据的资产化。采用"开箱即用"的使用方式，让用户无需再关心底层集群的搭建和运维，能极大提高数据开发和应用的效率。
